HTML

<html>
This identifies what sort of document the file is.
One should also set the lang attribute for SEO purposes.
<head>
Nothing in here shows up rendered in the webpage.
Typically contains <meta>, <title>, <link>, and <script >with <defer> set.
<meta>
Declare character set here.
<title>
The title of the window of the webpage goes here.
<link>
Contains reference to CSS style-sheet among other documents.
<script>
References Javascript file; user can enclose code inside here also.
<body>
Contains all rendered content.
<h1>
Contains the rendered title of the webpage.
<h[2-6]>
Contains the title of some section lesser than the element that encloses <h1>
<p>
Contains paragraphs of text.
<em>
Contains text meant to be emphasized.
