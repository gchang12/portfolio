List of Elements

HTML

<html>
This identifies what sort of document the file is.
One should also set the lang attribute for SEO purposes.
<head>
Nothing in here shows up rendered in the webpage.
Typically contains <meta>, <title>, <link>, and <script >with <defer> set.
<meta>
Declare character set here.
<title>
The title of the window of the webpage goes here.
<link>
Contains reference to CSS style-sheet among other documents.
<script>
References Javascript file; user can enclose code inside here also.
<body>
Contains all rendered content.
<h1>
Contains the rendered title of the webpage.
<h[2-6]>
Contains the title of some section lesser than the element that encloses <h1>
<p>
Contains paragraphs of text.
<em>
Contains text meant to be emphasized.

<a>
To link to a certain section of a webpage:
  1. Affix the desired section with an id attribute (e.g. my-id).
  2. In the tag, set the <href> attribute to "#", followed by the <id> value (e.g. <a href="#my-id">clickable text</a>)

If linking to a that webpage section from a different webpage:

<a href="/mysite/different-page#my-id">clickable text</a>

CSS

<rgb>
Is supported in most browsers.
Hue is represented by a color wheel (0..360).
Saturation and lightness are represented by percentages.
Implemented by the rgb function
Alpha-value specified as fourth argument, following a forward slash.
<hsl>
A model in which the addition of colors are additive.
Each value for red, green, and blue is represented by an integer from 0 to 255.
Implemented by the hsl function
Alpha-value specified as fourth argument, following a forward slash.
